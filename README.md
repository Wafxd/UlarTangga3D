#

V => 3D view

2x V => Side View

3x V => Normal View 2D

##

### In 3D View (V)

I => top rotation

K => bottom rotation

L => right rotation

J => left rotation

O => rotation turn to the right

U => rotation turn to the left

##

### In Side View (2x V)

arrow right left for rotate turn to the right or left

##

### In global

1 => change theme mode

2 => add lighting

3 => add more lighting (maybe)

##

### Gameplay

Press Space on ur kb to roll dice and enjoy the game
